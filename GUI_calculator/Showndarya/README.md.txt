# Calculator Application

> A simple calculator application built using C#

> More than one operator calculation is supported. The precedence order is Division-Multiplication-Addition-Subtraction

# Deployment Steps

#### Using code

> Open the .sln file in Visual Studio 2017

> Run the application

#### Using installer (Microsoft .NET Framework 4.6.1 required or higher)

> Download the [Zip folder for setup](https://drive.google.com/open?id=1gHiF0GRx4htemOQSTa4992zcOl3xyDfT) 

> Extract the contents to a folder.

> Run setup.exe
